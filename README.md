# vue-xiaod

## 业务需求
- 可视化展示数据
  - 视频的成交量
  - 用户总量
  - 订单总额
- 登录页
- 视频管理
  - 上传视频
  - 更新视频
  - 删除视频
  - 查看已有视频
- 用户管理
  - 更新用户管理
  - 删除用户
  - 新增用户
  - 权限管理
